
**适合**：

- ✅ 追求高收益

## 💡 为什么15分钟市场可行？

### 1. 快速对冲 ✅

```
你的观察：
  "随时看订单簿"
  "一次下小一点"
  "1U起步"

实现：
  - 1秒更新频率
  - 4个就触发对冲
  - 最大库存10个（5U）
  - 随时根据订单簿调整
```

### 2. 小资金友好 ✅

```
订单大小：
  - 每单 2 个（1 U）
  - 最大库存 10 个（5 U）
  - 不占用太多资金

优势：
  - 10-50 U 都可以做
  - 1U 起步门槛低
  - 可以频繁交易
```

### 3. 高频交易 ✅

```
一天交易轮数：
  - 16 小时 × 4 轮/小时 = 64 轮
  - 每轮 15 分钟
  - 资金快速周转

收益放大：
  - 虽然每轮收益小
  - 但轮数多
  - 总收益可能更高
```

### 4. 风险可控 ✅

```
严格限制：
  - 最大库存 10 个
  - 4 个就对冲
  - 日亏损 -3U 止损
  - 波动率 12% 上限

随时调整：
  - 看 1 秒刷新的订单簿
  - 根据市价快速调整
  - 及时对冲风险
```

## ⚠️ 风险提示

### 15分钟市场的风险

1. **价格快速变化**
   - 15分钟内可能反转多次
   - 需要快速反应

2. **最后5分钟**
   - 时间价值快速衰减
   - 价差收窄
   - 盈利机会减少

3. **流动性风险**
   - 某些时刻可能没有对手方
   - 订单簿可能很浅
   - 难以大单对冲

### 保护措施

```
策略自带保护：
  ✅ 最大库存 10 个（5U）
  ✅ 4 个就对冲
  ✅ 日亏损 -3U 止损
  ✅ 波动率 12% 限制
  ✅ 1 秒快速更新

额外建议：
  - 监控第一个15分钟轮次
  - 观察盈亏情况
  - 如果连续亏损，暂停
  - 不要在极端行情时运行
```
